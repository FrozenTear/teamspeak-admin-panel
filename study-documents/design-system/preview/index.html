Button / Input / Field / Tag / Toast / Modal / Empty state / Skeleton / Table — every primitive on one page.

Two-pane brand + form. Accent glow background. Setup callout when needsSetup.

Invalid-credentials banner above the form. Password cleared, focus returned.

Connect TS6 server. Test connection inline; advanced section progressive-disclosed.

Dashboard (happy)

KPI strip · bandwidth chart · connection card · activity feed. Full chrome.

Dashboard (empty)

Server selector

Dropdown / Menu primitive

Servers (list)
